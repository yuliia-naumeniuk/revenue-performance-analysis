# Revenue Performance Analysis
SQL analysis of revenue distribution by continent and device type with account and session metrics.

## Business Task
Analyze revenue distribution across continents and identify the contribution of mobile and desktop devices.
Additionally, calculate account and session metrics to better understand user activity.

## Tools
SQL, CTE, Joins, Aggregations, Window Functions, CASE WHEN

## Key Insights
- The **Americas generate the highest total revenue** among all continents.
- A significant portion of revenue comes from **desktop devices**, although mobile also contributes substantially.
- The Americas also have the **highest number of accounts and sessions**, indicating higher user activity compared to other regions.
